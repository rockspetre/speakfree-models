# speakfree-models

Signed model manifests and release artifacts for Speakfree.

The catalog currently supports English speakers practising Spanish or German. Release artifacts are downloaded from immutable upstream revisions, checked against pinned SHA-256 digests, signed with an Ed25519 key held in GitHub Actions secrets, and published as immutable GitHub Releases.

See [THIRD_PARTY_NOTICES.md](THIRD_PARTY_NOTICES.md) for model provenance and licensing.
