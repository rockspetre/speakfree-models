# speakfree-models

Signed model manifests and runtime artifacts for Speakfree.

The catalog supports English speakers practising Spanish or German with local speech recognition and tutor generation. Runtime releases are built from pinned upstream commits. Large official model files may be downloaded directly from immutable upstream revisions; every artifact is still size- and SHA-256-pinned by the Ed25519-signed catalog.

See [THIRD_PARTY_NOTICES.md](THIRD_PARTY_NOTICES.md) for model provenance and licensing.
